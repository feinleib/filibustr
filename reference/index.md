# Package index

## Voteview

Get data from [Voteview](https://voteview.com/data).

- [`get_voteview_members()`](https://feinleib.github.io/filibustr/reference/get_voteview_members.md)
  : Get data on members of Congress from Voteview
- [`get_voteview_parties()`](https://feinleib.github.io/filibustr/reference/get_voteview_parties.md)
  : Get data on congressional parties from Voteview
- [`get_voteview_rollcall_votes()`](https://feinleib.github.io/filibustr/reference/get_voteview_rollcall_votes.md)
  : Get data on congressional roll call votes from Voteview
- [`get_voteview_member_votes()`](https://feinleib.github.io/filibustr/reference/get_voteview_member_votes.md)
  : Get data on the votes of individual members of Congress
- [`get_voteview_cast_codes()`](https://feinleib.github.io/filibustr/reference/get_voteview_cast_codes.md)
  : Key to Voteview cast codes in individual member votes

## Senate.gov

Get data from [Senate.gov](https://www.senate.gov).

- [`get_senate_cloture_votes()`](https://feinleib.github.io/filibustr/reference/get_senate_cloture_votes.md)
  : Senate cloture motions and votes
- [`get_senate_sessions()`](https://feinleib.github.io/filibustr/reference/get_senate_sessions.md)
  : Start and end dates of Senate sessions

## Other research sources

Get data from [Legislative Effectiveness
Scores](https://thelawmakers.org) and [Harbridge-Yong, Volden, and
Wiseman (2023)](https://doi.org/10.1086/723805).

- [`get_les()`](https://feinleib.github.io/filibustr/reference/get_les.md)
  : Get Legislative Effectiveness Scores data
- [`get_hvw_data()`](https://feinleib.github.io/filibustr/reference/get_hvw_data.md)
  : Get replication data from Harbridge-Yong, Volden, and Wiseman (2023)

## Congresses and years

Convert between Congress numbers and years.

- [`congress_in_year()`](https://feinleib.github.io/filibustr/reference/congress_in_year.md)
  : Calculate the Congress number of a given year
- [`year_of_congress()`](https://feinleib.github.io/filibustr/reference/year_of_congress.md)
  : Get the starting year of a Congress
- [`current_congress()`](https://feinleib.github.io/filibustr/reference/current_congress.md)
  : Calculate the current Congress number

## Utilities

Not specific to political science research, but useful anyways!

- [`read_html_table()`](https://feinleib.github.io/filibustr/reference/read_html_table.md)
  : Scrape an online HTML table
